## شروع کار با مخزن .github

این مخزن شامل پوشه‌ها و فایل‌های ضروری برای ساختار سازمان کوشک هنر است:

- پوشه‌ها و زیرپوشه‌ها:
  - `profile/`
  - `brand-assets/`
  - `design-vault/`

- فایل‌های ضروری:
  - `README.md`
  - `CONTRIBUTING.md`
  - `CODE_OF_CONDUCT.md`
  - `ISSUE_TEMPLATE.md`
  - `PULL_REQUEST_TEMPLATE.md`

- فایل‌های نمونه برای Brand Assets:
  - `brand-assets/colors.md`
